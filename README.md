# AlgoVerse — DAA Learning Portal ⚡

AlgoVerse is a premium, guided step-by-step learning platform designed to help students master **Design and Analysis of Algorithms (DAA)**. Built with a focus on progressive learning, Algovise transforms static theory into an engaging educational journey.

![AlgoVerse Preview](https://github.com/PARIKSHIT-KUREL/AlgoVerse/raw/main/preview.png) *(Note: Add your own preview screenshot here)*

## 🚀 Key Features

- **Guided Learning Path**: A structured curriculum from fundamental complexity analysis to advanced Dynamic Programming.
- **Locked Progression**: Lessons are locked sequentially. Complete the "Knowledge Check" to unlock the next topic.
- **Interactive Checkpoints**: Test your understanding with built-in quizzes for every lesson.
- **Premium Dark UI**: A modern, responsive interface with a focus on readability and aesthetics.
- **Space & Time Insights**: Detailed metadata for every algorithm, including Big O notations.
- **Persistence**: Your progress is automatically saved to your browser's local storage.

## 🛠 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3
- **Logic**: Vanilla JavaScript (ES6+)
- **Typography**: Syne, Manrope, JetBrains Mono
- **Syntax Highlighting**: Prism.js

## 📖 Course Modules

1. **Performance Analysis**: Big O, Asymptotic Notations, Insertion/Bubble/Selection Sort.
2. **Divide & Conquer**: Binary Search, Max-Min, Merge Sort, Quick Sort.
3. **Greedy Method**: Dijkstra's, Fractional Knapsack, Job Sequencing.
4. **Dynamic Programming**: 0/1 Knapsack, TSP, Floyd-Warshall, Fibonacci.
